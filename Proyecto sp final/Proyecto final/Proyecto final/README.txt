MANUAL DE USUARIO

Este programa sirve para registrar datos de alumnos.
El programa solo se puede usar en windows.


Ignorar si la primera vez que se abre el archivo dice que no se encuentra un documento.

Al iniciar e2
l programa aparecera un menu, en este menu ingresas UNICAMENTE el numero del indice de la opcion que desees realizar.

En registrar alumno, lo primero que aparece es ingresar el nombre, puedes ingresar los nombres que quieras con espacios.

Lo siguiente es ingresar apellidos.

La siguiente es la Matricula, puedes ingresar una matricula o autogenerarla PERO solo ingresa numeros.

Continua con el correo electronico, puedes ingresar tu correoelectronico este debera tener, @ y .com.

Luego el telefono, este solo podra tener entre 8 y 12 caracteres y solo podras ingresar numeros

A continuacion las direcciones del alumno, Puedes ingresar numeros y espacios

Y por ultimo las calificaciones, deberas ingresar calificaciones del 0 al 100.

En buscar y modificar datos de un alumno, deberas ingresar la matricula del alumno que deseas buscar, y la modificacion es como el registro del mismo.

Mostrar lista de alumnos mostrara una lista de todos los alumnos registrados al momento.

Eliminar un alumno te permitira eliminar a un alumno ingresando la matricula. Te preguntara si estas seguro de borrarlo.

Y por ultimo la opcion de salir del programa, recuerda salir del programa para poder guardar tus avances.
